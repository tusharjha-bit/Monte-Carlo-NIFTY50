# Monte Carlo NIFTY50 Simulation

A Python-based Monte Carlo Simulation project to analyze possible future price paths of NIFTY50 using historical market data.

## Simulation Output

![Monte Carlo Simulation Result](monte_carlo_result.png)

## Features

- Historical NIFTY50 data analysis
- Monte Carlo price path simulation
- Probability distribution visualization
- Risk analysis

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- yFinance
